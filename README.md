
# 🏥 Healthcare Revenue Analytics Dashboard


##  Overview

This project presents an interactive Power BI dashboard analyzing hospital revenue across key medical services. It provides a comprehensive view of financial performance to support data-driven healthcare decisions.



##  Key Features

 Revenue Analysis by Medical Service
 Patient Segmentation
 Payment Type Analysis (Cash / Corporate)
 Doctor Specialty Insights
 Regional & Nationality Analysis
 Year-over-Year Growth Comparison
